# 🏥 Análise de Dados - Clínica Médica

Este projeto simula um cenário real de análise de dados em uma clínica médica, com o objetivo de extrair insights relevantes sobre consultas, médicos, pacientes e faturamento.

A análise foi realizada utilizando **SQL para exploração e tratamento dos dados** e **Excel para criação de indicadores e visualizações em dashboard**.

---

## 📊 Objetivo do Projeto

O objetivo deste projeto é analisar dados operacionais da clínica para gerar informações que auxiliem na tomada de decisão, respondendo perguntas como:

* Qual é o faturamento mensal da clínica?
* Quais médicos geram mais receita?
* Quais pacientes mais contribuem para o faturamento?
* Como as consultas estão distribuídas entre as cidades?

---

## 📈 Análises Realizadas

### Faturamento por Mês

Análise da evolução do faturamento da clínica ao longo do tempo para identificar tendências e variações mensais.

### Faturamento por Médico

Identificação dos médicos responsáveis pelo maior volume de receita na clínica.

### Pacientes com Maior Receita

Ranking dos pacientes que mais contribuíram para o faturamento total da clínica.

### Consultas por Cidade

Análise da distribuição geográfica das consultas realizadas.
